01 / PROFILE

"การสอนไม่ใช่การเติมน้ำให้เต็มถ้วย แต่คือการจุดประกายไฟแห่งการเรียนรู้ให้สว่างไสว"

PERSONAL PROFILE

ข้าพเจ้า นายพิชญะ นิวส์ นักศึกษาฝึกประสบการณ์วิชาชีพครู มุ่งมั่นในการพัฒนาทักษะการคิดเชิงคำนวณ (Computational Thinking) และทักษะการเขียนโปรแกรมแก่ผู้เรียน เพื่อเตรียมความพร้อมสู่การเป็นพลเมืองดิจิทัลที่มีคุณภาพในอนาคต ด้วยวิธีการสอนแบบ Active Learning ที่บูรณาการทฤษฎีการสร้างความรู้ (Constructivism)

ED

ประวัติการศึกษา

ครุศาสตรบัณฑิต (ค.บ.) สาขาวิชาคอมพิวเตอร์ศึกษา มหาวิทยาลัยเทคโนโลยีราชมงคล

SK

ทักษะและความเชี่ยวชาญ

Python, Web Tech (HTML/CSS/JS), Algorithms, Computational Thinking, Block-based Programming

02 / THE SCHOOL

SCHOOL CONTEXT

โรงเรียนเตรียมอุดมศึกษาเป็นสถานศึกษาขนาดใหญ่พิเศษ จัดการเรียนการสอนในระดับชั้นมัธยมศึกษาตอนปลาย มุ่งเน้นการพัฒนาความเป็นเลิศทางวิชาการและคุณธรรม

01

วิสัยทัศน์

ผู้นำความเลิศทางวิชาการ มาตรฐานสากล พัฒนาสังคมอย่างยั่งยืน

02

สภาพแวดล้อมการเรียนรู้

ห้องปฏิบัติการคอมพิวเตอร์ที่ทันสมัย พร้อมระบบอินเทอร์เน็ตความเร็วสูง
